# HOME 0.1

**FA（ファクトリーオートメーション）制御エンジニア**のために設計された、特化型生産性向上ワークスペースです。三菱電機、キーエンス、オムロンなど、異なる制御システムに必要な多様な自動化ツールやネットワーク設定の管理を効率化します。

## 🏭 FAエンジニア向け設計

制御エンジニアは、特定のソフトウェアツール（PLCプログラミング、センサー設定など）や独自のネットワーク設定を必要とする異なるシステム間を行き来することがよくあります。**HOME 0.1**は、この「コンテキスト切り替え」の問題を解決します。

*   **プロジェクトベースのツール管理**:
    *   **三菱電機**のシステムで作業しますか？プロジェクトを作成し、**GX Works**、**GT Designer**、および特定のセンサーツールのショートカットを追加します。
    *   **キーエンス**のシステムに切り替えますか？プロジェクトを切り替えるだけで、ワークスペースが即座に更新され、**KV Studio**や関連マニュアルが表示されます。
*   **ネットワーク切り替えの自動化**:
    *   コントロールパネルから手動でIPアドレスを変更する必要はもうありません。
    *   プロジェクトごとにネットワーク設定を定義します（例：ラインA用 `192.168.3.100`、ラインB用 `192.168.0.10`）。
    *   プロジェクトを切り替えると、**HOME 0.1**は自動的にPCのネットワークアダプターをターゲットシステムに合わせて再構成します。

## 機能

### 📁 プロジェクト管理
- **作業の整理**: 個別のプロジェクトを作成して、作業を分けて管理します。
- **プロジェクト固有データ**: 各プロジェクトには、独自のショートカット、メモ、ネットワーク設定があります。
- **ネットワーク切り替えの自動化**: プロジェクトを切り替える際に、システムのIPアドレスとゲートウェイ設定を自動的に変更します（管理者権限が必要です）。

### 🔗 スマートショートカット
- **クイックアクセス**: ファイル、フォルダ、またはURLへのショートカットを追加して簡単にアクセスできます。
- **ドラッグ＆ドロップインポート**: ファイルやフォルダをアプリにドラッグするだけで、ショートカットを作成できます。
- **ドラッグ＆ドロップ並べ替え**: アイテムをドラッグして、ショートカットグリッドを簡単に並べ替えることができます。
- **グローバルショートカット**: よく使うアイテムをサイドバーにピン留めして、すべてのプロジェクトからアクセスできます。

### 📝 統合メモ
- **リッチテキストエディタ**: 書式設定（見出し、リスト、太字、斜体など）をサポートしたメモを作成できます。
- **インポート＆エクスポート**: テキスト/Markdownファイルをインポートしたり、メモをMarkdownとしてエクスポートしたりできます。
- **自動保存**: 入力中にメモは自動的に保存されます。
- **ミニ＆拡大表示**: クイックアクセスのミニビューと、全画面のモーダルエディタを切り替えることができます。

### 📅 カレンダー統合
- **Googleカレンダー**: アプリ内でスケジュールを直接確認できます。
- **ローカライズ**: 日付形式やナビゲーションを含む、完全な日本語サポート。
- **視覚的な手がかり**: 週末と今日の日付を明確に区別するスタイリング。
- **メモ**: カレンダーの日付に毎日のメモを追加できます。

### ⚙️ カスタマイズと設定
- **モノクロテーマ**: 洗練された、気が散らないモノクロデザイン（ライト/ダークモード）。
- **言語サポート**: 英語と日本語のインターフェースを切り替えることができます。
- **グローバルネットワーク設定**: アプリケーションによって管理されるデフォルトのネットワークインターフェース（イーサネット、Wi-Fiなど）を設定します。
- **データ管理**: バックアップ用にワークスペースデータ全体をエクスポート/インポートできます。

## 始め方

### 前提条件
- Node.js (v16 以上)
- npm または yarn

### インストール

1.  リポジトリをクローンします:
    ```bash
    git clone <repository-url>
    ```
2.  プロジェクトディレクトリに移動します:
    ```bash
    cd HOME_0.1
    ```
3.  依存関係をインストールします:
    ```bash
    npm install
    ```

### アプリの実行

開発サーバーを起動します:
```bash
npm run dev
```

### 本番用ビルド

OS用のアプリケーションをビルドします:
```bash
npm run build
```
